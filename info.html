Willkommen bei inTwit im BMF, einem Spiel, das Ihr Wissen über das deutsche Finanzministerium testen soll. Wie gut kennen Sie das schöne Gebäude und seine Bundesregierung?

Wie wird gespielt?

Das Spiel imitiert das berühmte „Wer wird Millionär“. Diesmal: Wer kennt das BMF?

Sie erhalten ein paar Fragen mit den entsprechenden Optionen (a, b, c, d). Sie müssen die richtige Antwort auf jede Frage auswählen. Bitte beachten Sie, dass auf jede Frage nur eine Antwort richtig ist. Wenn deine Antwort richtig ist, wird automatisch die nächste Frage angezeigt und deine Punktzahl erhöht sich.
